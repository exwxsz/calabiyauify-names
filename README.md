<h1 align="center">卡拉彼丘程度计算器</h1>


一个超级可爱的卡拉彼丘（Calabiyau）程度计算器！通过输入昵称，计算你有多像卡丘猫娘喵~ ✨

## 🌍 在线预览：<img align="right" src="public/aika3.png"  width="64">


[https://calabiyauname.dpdns.org/](https://calabiyauname.dpdns.org/)




## 🌟 功能特性 <img align="right" src="public/title.png"  width="64">

- **智能计算**：基于多重哈希算法和特征加权，精准计算你的卡丘程度
- **关键词检测**：包含卡丘文化关键词直接100%喵！
- **等级系统**：从萌新到顶级猫娘，10个等级评价
- **可爱界面**：粉色气泡风格，支持暗色模式
- **互动效果**：
  - 点击屏幕产生喵喵动画
  - 点击标题播放音效和切换图片
  - 动态评论和角色图片展示
- **喵言喵语**：内置大量经典喵语数据，可复制喵言喵语发在卡丘公屏上喵~

## 🛠️ 技术栈 <img align="right" src="public/xnm1.png"  width="64">

- **前端框架**：Vue 3 
- **构建工具**：Vite
- **样式**：Tailwind CSS + PostCSS

## 🚀 本地部署 <img align="right" src="public/mi1.png"  width="64">

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 📖 使用说明 <img align="right" src="public/mi3.png"  width="64">

1. 打开应用，输入你的昵称
2. 点击"计算卡拉彼丘程度"按钮
3. 查看你的分数、等级评价和对应图片
4. 享受可爱的喵喵互动效果！


## 🎨 界面特色 <img align="right" src="public/aika4.png"  width="64">

- **响应式设计**：适配各种设备
- **暗色模式**：自动检测系统偏好，支持手动切换
- **动画效果**：平滑过渡和可爱动画
- **音效支持**：点击音效和背景音乐
